Get Started Documentation

Ease gives you the power of modern frameworks using the tools built right into the browser.

No need to set up a build enviroment, no need to learn a new language or syntax.

Just write your code and ship it.

Since Ease is built on web standards, you can use it anywhere you can use HTML, CSS, and JavaScript. It plays nicely with other frameworks and libraries, can be used to build isolated components, or full websites. Its modular design lets you pull in features you want and skip the ones you don't.

With very little to learn, anyone familiar with HTML can get started with Ease.

Load the Library

Start by adding the necessary script tags to your HTML file.

script typemodule srchttps://unpkg.com/@easedotjs/corescript
script typemodule srchttps://unpkg.com/@easedotjs/componentsscript

Load a Component

Once you've loaded the library, you can load a component from an HTML document.

link relcomponent namens-hello hrefcomponents/hello.html

Use the Component

Once you've loaded the component, you can use it in your HTML.

ns-helloHello World!ns-hello

Components can be complex, interactive isolated elements with their own styles, scripts, and templates - or simple, static elements that just render content.

Want to feel the nolstagia of the 90s? Load in jQuery and use it in your Ease components. Need to fetch data from an API? Use the native Fetch API. Want to use a modern library like React or Vue? No problem. Ease plays nicely with others.

Everything you need is here because everything you need is already in the browser.

Ease packages include...

@easedotjs/core - The core library
@easedotjs/components - Loads components from HTML documents
@easedotjs/reactive-dom - Make the content of your components reactive
@easedotjs/router - Make your sites load additional pages without reloading
